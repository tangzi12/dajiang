# 第十六章 孤儿

大撤退到了第二十八天。

郑璧成在码头上发现了那群孤儿。不是"发现"——是差点踩到一个。一个大概四岁的男孩蹲在栈桥角落里，缩成一团，脸上被干了的鼻涕和灰糊成了一个硬壳，正用一根从货箱上剥下来的竹篾在地上画圈。旁边还有十来个孩子。最大的大概十一二岁，是个女孩，已经瘦得脱了相。她把其他几个更小的圈在身后——让他们往墙和板条箱缝里挤。最小的一个一直被一个半大孩子抱在怀里——大概两岁，嘴张着，没有哭，哭声已经干了。孩子们身上穿着规整的棉布衫——但不是本地的。棉衫的襟口绣着极细的蓝色十字线，有几个领后还缝着一小条名字布标，针脚细密。只是如今都灰扑扑的，被泥点子和草屑打了满背。

他们是从南京保育院一路撤下来的。保育院在南京陷落前带着孩子们过了江，坐船到汉口，又从汉口坐了十天的敞篷驳船到宜昌。一路上他们从一个穿灰布袄的女老师带着几十个孩子——出发时她带着名单，在汉口号了一遍名字，还从名单上划掉了生病的两个孩子。到了宜昌以后，老师在空袭里被弹片击中后腰，抬去了临时救护站。她临走之前用最后一丝力气把背上抱着的婴儿递给年纪最大的女孩，说了一句"不要散"。孩子们等船等了四天。没有人管他们——码头上每个人都在抢运机器。机器不哭。机器不会饿。孩子们饿。但他们的哭声被码头上的汽笛、号子和防空警报盖得干干净净。

郑璧成数了数——十四个。他一眼认出领头那个女孩——她的眼睛和当年的他自己一模一样。那种在很窄的空间里待惯了的人才会有的直。他把工作交给副手，自己往调度棚跑。调度棚是用装货的木箱拼起来的一个临时雨棚，顶上搭着画有日文假名的旧帆布。卢作孚正端着一碗泡酱油拌的冷饭——已经放了半天，酱油在上面凝了一层薄油皮。

"有一群保育院的孤儿，等了几天都没人安排。再不给船——下一波空袭一到，怕就散了。"

卢作孚把碗放下——不是放，是往木箱上一顿。"还有多少位置？"

"下一班民和轮还有三十个人的容量。但我们已经答应了给兵工厂运那批炮弹——五百箱，已经绑了一半的货。"

"炮弹等。"卢作孚的嗓子里卡着一些什么东西——不是痰，是太多天没有好好喝过水了，喉咙壁已经干得发粘。"孩子不等。"

郑璧成看着那碗冷饭，没有立刻动。他看着卢作孚。他心里想的是——五百箱炮弹，够一个步兵团打一次阻击。但他说不出口。因为他知道卢作孚的心里已经称过那杆秤了——一头是炮弹，另一头是那群穿着绣了十字线棉衫的孩子。炮弹是打日本人的。孩子——是日本人打完以后活下来的人。

他没有再问。转身跑向码头。他怕自己再多站一会儿——也会在调度棚里哭出来。

民和轮的船长接到调度令的时候——船已经装了一大半炮弹。木箱堆了半个统舱，棕绳只绑到第四层。副手建议把货卸下来重新堆，但是重堆最少要半天。船长把烟斗从嘴里拔出来，看着码头另一端那群还没走完跳板的孩子——最小的自己不会走，是由那个被击中的女老师背上来的最后一个婴儿，此刻在领头女孩怀里哭。他把烟斗在铁栏杆上磕干净。"不卸。孤儿坐货堆上。每箱炮弹中间塞一个娃娃。"

孤儿们被船员一个一个抱上船。最小的那个——两岁的——被茶房老李抱过去的时候吓得往郑璧成怀里躲，他抓住郑璧成的衣领不放。郑璧成把他交给老李，孩子的手从衣领上滑开的那一刻，郑璧成摸到了小拳头里硌人的东西——是一粒包着糖纸的冰糖，已经化了一半，融出一些砂糖沾在手心。老李看起来老——其实不过四十岁，他自己有两个儿子在重庆，但三年没回家。他把孩子抱进船舱的时候，统舱里的炮弹箱已经堆到了齐肩高。他找了一处相对平整的弹药箱顶面，从自己铺位上抽出一件旧棉衣垫上去，再把孩子放稳。然后他从衣襟里摸出最后几粒喂嘴的冰糖——战时航程不准带私人物资，这冰糖是他自己攒下来的，包在糖纸里都快化了——塞进孩子嘴里。两岁的孩子含着冰糖，发现甜，忽然不哭了。老李松了口气，又打开了自己的水壶盖子——给孩子看里面晃荡的凉白开，像在表演一个戏法。

"船去重庆。重庆不打仗。"老李对孩子说。语气就像平时在统舱里给乘客送热茶。

统舱里全是孩子。十四个，蜷在不同的空隙里。民和轮的统舱本来就不大——现在一半是炮弹，一半是孤儿。船上其他几个茶房把棉被从茶具柜里抱出来，一个一个铺在炮弹箱的棕绳之间。被子上的字还在——"民生为天"。

那个年纪最大的女孩上船最晚。她站在跳板尽头，回头看了一眼宜昌——码头上还在装货，火光和汽笛声混在一起。她怀里还抱着那个最小的婴儿，婴儿已经吃完了所有的冰糖——睡着之后含住的糖纸角慢慢从唇缝上松开。郑璧成站在跳板下面对女孩说："上船——我会想办法把你们老师转到重庆的医院。"

"不用想办法。她已经走了。我听见护士说的。"女孩的声音很平。不是不怕。是当了太久的家长之后忘记了怎么要求安慰。

汽笛响了。民和轮离开宜昌码头，逆流而上，驶入夜幕。郑璧成在船后站了一阵。他的眼眶里有些什么——但他没有让它们溢出。

这艘船安全到达了重庆。十四个孤儿全部活了下来。几年之后，那个领头女孩考进了北碚的兼善中学——卢作孚自己办的那所学校。她在作文里写到过来宜昌的那一晚——写到民和轮上炮弹箱之间铺着的棉被，写到茶房老李把最后几颗半化的冰糖划成碎末，轮流给每个孩子舌尖放了一点。她在作文最后写了一段话。

"我们这些人——不是在平安里长大的。是在一条叫民生的船上。"

那批被推迟运送的五百箱炮弹两天后由下一班船运到前线——比原定到达时间迟了将近四十八个小时。负责接弹药的作战参谋在回执单上打了一个问号。押运的民生公司副手在回执栏里用铅笔写了一句:"此位先让孤儿。炮弹仍到，不误交火。"那个问号后来在档案里一直留着，但回执没有被驳回。

卢作孚从调度棚转过身时仍然端着那碗已经凉透的酱油饭。码头上的风把帆布棚角一下一下撩起来打在箱子角上，他嚼着冷饭，心里还在默算下一班船的吃水线。
