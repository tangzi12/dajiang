# 大江 — 24集电视剧开发案

> 卢作孚原型 · 民族航运商战 · 长江史诗

## 项目简介

《大江》是一部24集商战史诗电视剧开发案，以"中国船王"卢作孚（1893-1952）的真实人生为蓝本，讲述一个合川穷小子如何凭一艘70吨旧船，在长江上与太古、日清等外资航运势力周旋，最终在宜昌大撤退中把中国工业命脉抢运入川。

本案借鉴经典民族商战剧的叙事传统：熟人社会里的商战、技术细节里的胜负、人情债里的生死账、个人命运与家国大势的互相碾压。但《大江》的核心不是致敬，而是卢作孚自己的故事：一条江，一群跑船的人，一个把"民生"两个字当成命来做的人。

故事从1893年四川合川嘉陵江边一个卖麻布的小贩家庭开始，到1952年骨灰撒入长江结束。六十年。一条江。一个人。

**核心看点：**
- 1938年宜昌大撤退——最紧急的40天抢运人员和关键物资入川，后续转运保存了中国工业命脉
- 袍哥堂口一人一坛酒换朝天门码头
- 英国太古、日本日清两大殖民航运巨头的商战围剿与反杀
- "民生公司的船，不为日本人运一颗螺丝钉"
- 一个实业家最后留下的不是豪言壮语，而是账、家具、股票，和一盏不能灭的灯

## 创作方法

1. 以卢作孚真实生平为主干，区分"有史料依据""合理戏剧推演""纯虚构合成人物/场景"
2. 借鉴经典民族商战剧的叙事方法，但避免直接复刻桥段
3. 把航运技术、码头规矩、票价战、调度账本写成观众能看懂的戏
4. 强化人物代价：卢作孚不是圣人，他严苛、亏欠家庭、也会做残酷选择
