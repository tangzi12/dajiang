# 大江 — 24集电视剧剧本

> 卢作孚原型 · 《大染坊》叙事技法的长江版

## 项目简介

《大江》是一部24集商战史诗电视剧剧本，以"中国船王"卢作孚（1893-1952）的真实人生为蓝本，完整复刻了经典商战剧《大染坊》的叙事技法——"熟人社会商战""技术细节戏剧化""饭桌三段式对话""底层逆袭×家国悲剧"。

故事从1893年四川合川嘉陵江边一个卖麻布的小贩家庭开始，到1952年骨灰撒入长江结束。六十年。一条江。一个人。

**核心看点：**
- 1938年宜昌大撤退——40天抢运100万吨物资、150万难民，中国版敦刻尔克
- 袍哥堂口一人一坛酒换朝天门码头
- 英国太古、日本日清两大殖民航运巨头的商战围剿与反杀
- "民生公司的船，不为日本人运一颗螺丝钉"

## 创作方法

1. 逐章精读《大染坊》原著PDF，提取陈杰的8种核心叙事秘技（鱿鱼温度计、冷砣保密、饭桌三段式、分着卖、先跪后赢等）
2. 将每种技法平移到卢作孚的历史语境中
3. 所有商战情节和历史事件均以真实史料为锚点
