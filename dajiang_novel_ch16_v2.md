# 第十六章 孤儿

大撤退第二十八天。郑璧成在码头上发现了那群孤儿。不是特意去找——是他蹲在栈桥边核对装船单时差点踩到一个四岁男孩。男孩缩在栈桥角落里，脸上被干鼻涕和灰糊成了硬壳，正用一根从货箱上剥下来的竹篾在地上画圈。旁边还有十来个孩子，最大的十一二岁，是个女孩，瘦得快脱了相。她把更小的孩子圈在身后——让他们往墙和板条箱缝里挤。最小的一个被女孩抱在怀里——大概两岁，嘴张着，没有哭，哭声已经干了。

孩子们是从南京保育院一路撤下来的。出发时带队的女老师领着名单，在汉口渡口还划掉了两个发烧的孩子。到了宜昌以后女老师在空袭里被弹片击中后腰，抬去了临时救护站。临走前她用最后一丝力气把背上的婴儿递给年纪最大的女孩，说了一句"不要散"。孩子们在码头上等了四天。没有人来安排他们——码头上每个人都在抢运机器。机器不哭。机器不会饿。但孩子们会。

郑璧成跑回调度棚。卢作孚正端着一碗泡酱油的冷饭——酱油在饭面上凝了一层薄油皮。

"一群保育院的孤儿。等了几天没人安排。再不给船——下一波空袭一到怕就散了。"

"还有多少位置？"

"下一班民和轮还有三十个人的容量。但已经答应给兵工厂运那批炮弹——五百箱。"

卢作孚把碗往木箱上一顿。"炮弹等。孩子不等。"

郑璧成转身跑向码头。民和轮的船长收到调度令时船已装了大半炮弹。他把烟斗从嘴里拔出来:"不卸。孤儿坐货堆上。每箱炮弹中间塞一个娃娃。"

孤儿们被船员一个一个抱上船。最小的那个两岁婴儿被茶房老李抱过去时吓得往郑璧成怀里躲。老李从衣襟里摸出最后几粒喂嘴的冰糖——他自己攒的，包在糖纸里都快化了——塞进孩子嘴里。孩子含着糖忽然不哭了。

那个最大的女孩上船最晚。她站在跳板尽头回头看了一眼宜昌——码头上还在装货，火光和汽笛声混在一起。郑璧成站在跳板下面对她说会想办法把她们的老师转到重庆的医院。女孩的声音很平——"她已经走了。我听见护士说的。"汽笛响了。她抱着婴儿转身走进船舱。

民和轮安全到达重庆。船上十四个孤儿全部活了下来。几年后那个领头女孩考进了北碚兼善中学，在作文里写到宜昌的那一晚——写到炮弹箱之间铺着的棉被，写到茶房老李把最后几颗半化的冰糖敲碎分给每个孩子。作文最后一句是:"我们这些人——不是在平安里长大的。是在一条叫民生的船上。"

那批被推迟的五百箱炮弹两天后由下一班船运到前线。押运的民生副手在回执栏里只写了一行铅笔字:"此位先让孤儿。炮弹仍到。不误交火。"

（第十六章 完 · 第十七章「何老头」——纤夫在寒冬中拉纤三十九天，老何在最后一天悄然离世）
